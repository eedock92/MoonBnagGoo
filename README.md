# MoonBangGoo
react-redux-node.express-mongodb를 이용한 온라인 쇼핑몰
디자이너의 굳즈, 장난감 판매

주요 기능 
 
 jwt인증 방식 로그인, 회원가입, 
 상품 장바구니 추가, 상품 검색,
 paypal 결제, 회원정보 변경
 
 관리자 : 상품등록, 삭제, 수정, 회원 등록, 수정
